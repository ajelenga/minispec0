GenCode est la génération du code Java à partir d'un fichier XML.
GenCodeTest est le test.

Syntaxe abstraite : Type
Les différents types sont Col (liste, hash Map ...) et Type Simple (integer, String ...).
Chaque element possède un metamodel (Attribute, Entuty et Model)
un attribut à un type col où Type simple.


XmlAnalyser et XmlSerialiser sont fonctionnels
